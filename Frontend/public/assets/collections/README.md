Put images for each category in the respective subfolders.

Naming convention:
- cover.jpg — primary image used by the site (recommended size: 1200x1600 or similar portrait)

Subfolders:
- Earrings/
- Lockets/
- Bracelets/
- Pendants/
- Combo/

Upload your images using the above names and the site will automatically use them from `/assets/collections/<Category>/cover.jpg`.
